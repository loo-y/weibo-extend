## Weibo Extend
微博扩展插件

********** 该插件仅用于学习和交流 **********

### 实现功能
1. 被对方拉黑之后，仍可查看其主页 (✔️)

2. 在拉黑某人之后，仍可查看其主页 (✔️)

3. 批量拉黑某条评论的点赞用户 (✔️)

4. 批量拉黑某条微博的点赞用户 (TODO)
